# NapCat 插件 - @提及守卫

监测指定群聊中非白名单用户@机器人时自动禁言30分钟。

## 功能

- 监控指定群聊（1107201723, 893387793, 170874625, 1033811323）
- 检测是否@了机器人
- 非白名单用户@机器人自动禁言30分钟
- 白名单用户不受限制

## 安装

1. 将插件文件夹复制到 NapCat 的插件目录
2. 重启 NapCat 或热重载插件
3. 在 NapCat WebUI 中进入插件配置并填写参数

## WebUI 配置项

- `监控群聊`（`monitoredGroups`）：需要启用守卫的群号列表
- `白名单用户`（`whitelist`）：这些用户 @ 机器人不会被禁言
- `禁言时长（分钟）`（`muteDuration`）：默认 30 分钟

如果不填写，将使用内置默认值：
- 监控群聊：`1107201723, 893387793, 170874625, 1033811323`
- 白名单：空
- 禁言时长：30 分钟

## 注意事项

- 机器人需要有群管理员权限才能禁言用户
- 白名单用户不受禁言规则限制
- 机器人无法禁言群主和管理员
