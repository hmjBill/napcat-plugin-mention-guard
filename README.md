# NapCat 插件 - @提及守卫

监测指定群聊中非白名单用户@机器人时自动禁言30分钟。

## 功能

- 监控指定群聊（1107201723, 893387793, 170874625, 1033811323）
- 检测是否@了机器人
- 非白名单用户@机器人自动禁言30分钟
- 白名单用户不受限制

## 安装

1. 将插件文件夹复制到 NapCat 的插件目录
2. 重启 NapCat 或热重载插件
3. 在配置中添加白名单用户（可选）

## 配置

编辑 `index.ts` 中的配置区域：

```typescript
// 受监控的群聊列表
const MONITORED_GROUPS = [
  "1107201723",
  "893387793", 
  "170874625",
  "1033811323"
];

// OneBot 白名单用户列表
const ONEBOT_WHITELIST = [
  "123456789",  // 添加白名单QQ号
  "987654321"
];

// 禁言时长（秒）
const MUTE_DURATION = 30 * 60;  // 30分钟
```

## 注意事项

- 机器人需要有群管理员权限才能禁言用户
- 白名单用户不受禁言规则限制
- 机器人无法禁言群主和管理员
